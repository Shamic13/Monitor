# Monitor
Monitor the temperature, workload, and consumption of your computer components

RU: 
Аппаратная часть: 
Плата Arduino UNO
Дисплей 16x2
USB-кабель (USB A - USB B)

Программная часть:
Python 3.7+ с библиотеками:
  psutil - системная информация
  wmi - доступ к датчикам Windows
  gpustat - мониторинг видеокарт
  pyserial - связь с Arduino
  Arduino IDE для прошивки
  OpenHardwareMonitor для доступа к датчикам
  
Сборка аппаратной части
1. Установите LCD Keypad Shield на Arduino UNO
2. Подключите USB кабель к компьютеру
3. Убедитесь, что Arduino определилась в системе

Перед началом использования обязательно установите и откройте утилиту OpenHardwareMonitor (ссылка - https://github.com/HardwareMonitor/OpenHardwareMonitor/releases), без нее не будут показываться температуры на вашем дисплее. Она должна постоянно быть открытой, либо находиться в трее
Программа рассчитана на дисплей LCD 16x2 с кнопками (название - LCD Keypad Shield)
Для запуска программы подключите схему к компьютеру, скачайте файл ардуино, откройте его и при необходимости измените порт COM на нужный, затем скачайте файл пайтона и запустите его

На LCD Keypad Shield доступны 5 кнопок:

Кнопка	 Функция	       Описание
SELECT	 Авторежим	     Вкл/Выкл автоматическое переключение режимов
UP	     Режим памяти	   Переключить на отображение памяти
DOWN	   Режим CPU/GPU	 Переключить на отображение процессора/видеокарты
LEFT	   Статус	         Показать статус подключения
RIGHT	   Информация	     Показать информацию о системе
